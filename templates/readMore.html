Welcome To PMD Project 2020

Using deep learning concepts to practically aim at remote sensing tasks. Remote Sensing plays a crucial role in mapping of land use/land cover.It is the recent technology to monitor land resources from space and evaluate the land cover changes.Spectral similarity between different land use/land cover classes are quite common when it comes to satellite imagery datasets.Therefore,automatic classification of heterogenous landscape is challenging to get accurate good classification accuracy.

In low and medium-resolution remote sensing images,the problem is that different objects may share the same spectral-response curve but,the same object may have different spectral response curve thus making the classification model quite complex at times.

To properly teach a machine how to classify a given set of satellite imagery data provided with certain land-classes is the crux of our work. In recent times, it is still vague as to what exact set of methodologies and transformations or a good permutation and combination of both: classifiers, features would be ample for a given set of problem. Ever since the rising of Machine Learning where instead of the traditional approach where, our machine is fed with data and programs produces a specific desired output; we allow the machine to learn on its own without being “explicitly programmed”;technologies have advanced vastly. There has also been observed an uprising against “hand-crafted” solutions in areas like signal/image processing,computer vision, and machine learning.

Recently the deep convolutional neural networks(CNNs) which are specialized kind of neural network for processing data that has a known, grid-like topology such as time-series(1D grid),image data (2D grid)etc. In this paper, we propose a self-trained model which would classify various landcovers amongst one of the six classes: barren-land,forests,snow-covered land,water bodies and grass-lands
